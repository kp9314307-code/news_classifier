# Программа для рубрикации новостей

## Описание
Программа автоматически определяет тематическую рубрику новостного текста. 
Пользователь указывает имя текстового файла, программа анализирует его содержимое 
и относит новость к одной из шести рубрик: Спорт, Политика, 
Наука и технологии, Культура или Общество.

## Как работает
1. Пользователь вводит имя файла с текстом новости
2. Программа читает файл и передаёт текст на обработку
3. Библиотека spaCy токенизирует текст-разбивает его на отдельные слова
4. Каждое слово приводится к начальной форме (лемматизация): например, 
   "матча" → "матч", "победили" → "победить"
5. Полученные леммы сравниваются с заранее составленными списками 
   ключевых слов для каждой рубрики
6. Рубрика с наибольшим числом совпадений определяется как результат
7. Программа выводит текст новости, определённую рубрику и статистику 
   совпадений по всем рубрикам

## Структура проекта
- news_classifier.py -основной файл программы
- news_sport.txt -пример новости про спорт
- news_politics.txt -пример новости про политику
- news_culture.txt -пример новости про культуру
- news_science.txt -пример новости про науку

## Технологии
- Python 3.11
- Библиотека spaCy-токенизация и лемматизация текста
- Языковая модель ru_core_news_sm-обработка русского языка

## Установка

pip install spacy

python -m spacy download ru_core_news_sm

## Запуск

python news_classifier.py

После запуска программа попросит ввести имя файла с новостью, например:

news_sport.txt

## Пример работы
Входной файл содержит текст:
"Российская сборная по хоккею одержала победу в финальном матче чемпионата мира."

Программа выводит:
РУБРИКА: Спорт

СТАТИСТИКА СОВПАДЕНИЙ:
Спорт            8 совп.  
Политика         0 совп.
Экономика        0 совп.
