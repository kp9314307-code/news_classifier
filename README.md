# Программа для рубрикации новостей

## Описание
Программа читает текстовый файл с новостью и определяет её рубрику: Спорт, Политика, Экономика, Наука и технологии, Культура или Общество.

## Как работает
1. Пользователь вводит имя файла с новостью
2. Программа токенизирует текст с помощью библиотеки spaCy
3. Каждое слово приводится к начальной форме (лемматизация)
4. Леммы сравниваются с ключевыми словами каждой рубрики
5. Рубрика с наибольшим числом совпадений-результат

## Установка
pip install spacy


python -m spacy download ru_core_news_sm

## Запуск
python news_classifier.py

## Технологии
- Python 3.11
- Библиотека spaCy
